# 🌊 Atlantis Front-end

Sistema front-end desenvolvido em React utilizando Vite para gerenciamento e navegação da aplicação Atlantis.

---


**Senha:**
```txt
123456
```

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Motivo |
|------------|---------|
| Vite | Dev server ultrarrápido e build otimizada |
| React 18 | Construção da interface |
| React Router DOM v6 | Navegação SPA sem recarregamento |
| Lucide React | Ícones SVG leves e modernos |
| CSS puro com variáveis | Controle total de tema e estilos |
| Google Fonts (Cormorant Garamond + Jost) | Tipografia da aplicação |

## 🚀 Como executar o projeto

### 1. Clone o repositório

```bash
git clone URL_DO_REPOSITORIO
```

---

### 2. Entre na pasta do projeto

```bash
cd frontend
```

ou caso tenha clonado a pasta principal:

```bash
cd ATIV-ATLANTIS/frontend
```

---

### 3. Instale as dependências

Execute:

```bash
npm install
```

ou:

```bash
npm i
```

Isso instalará todas as dependências presentes no arquivo `package.json`.

---

### 4. Inicie o servidor de desenvolvimento

Execute:

```bash
npm run dev
```

---

### 5. Abra o navegador

Após iniciar o projeto, o terminal exibirá algo parecido com:

```bash
VITE vX.X.X ready in XXX ms

➜ Local: http://localhost:5173/
```

Abra:

```txt
http://localhost:5173
```

---

## 📦 Scripts disponíveis

Iniciar projeto:

```bash
npm run dev
```

Gerar build de produção:

```bash
npm run build
```

Visualizar build localmente:

```bash
npm run preview
```
